---
title: "File to Text"
description: "Reads a document format input and extracts its textual content."
---

The `File to Text` module is a data module that can read a document and extract all its textual content.

You can use this module to transform any document of `PDF`, `DOCX`, `TXT` into text.

The `File to Text` module has **one input** and **one output**:

- **Input**:
  - One input of type `File`
- **Output**:
  - `X` number of Text outputs
