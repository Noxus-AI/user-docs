---
title: "Download File"
description: "Downloads a file from a URL"
---

The `Download File` is a data node that allows you to download files in your workflow, and consequently you can manipulate it's data. Consider using a  `File to Text` node after it.

The `Download File` module has **one input** and **one output**:

- **Input**:
  - One input of type `Text`
- **Output**:
  - One output type `File`
