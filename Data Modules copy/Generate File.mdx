The `Generate File` module is a data module that can convert text and images into a file.

You can use this module to transform your AI-generated text into a file of `PDF`, `DOCX`, `TXT`, `CSV` and `HTML` types

The `Generate File` module has ** variable inputs** and **one output**:

- **Input**: inputs can be of type `text` and `image`; you can add them in any specific to structure the document as you see fit
- **Output**: `File`, the output is of a file type in the specified format
